---
title: Installation
description: How to install dependencies and structure your app.
---

<Callout>
  **Note**: We have the exact same installation process as
  [shadcn/ui](https://ui.shadcn.com/docs/installation).
</Callout>

<Steps>

<Step>Create a project</Step>

Run the `init` command to create a new Next.js project or to setup an existing one:

```bash
npx shadcn@2.8.0 init
```

<Callout
  title="Using Shadcn/UI v2.8.0 Recommended"
  className='mt-6'
>
  Version 2.9.0 of Shadcn/UI includes a minor issue affecting the vendor registry.  
  For now, we recommend using v2.8.0 to ensure expected behavior.  
  You can follow the issue [here](https://github.com/shadcn-ui/ui/issues/7849).
</Callout>

<Step>Add components</Step>

You can now start adding components to your project.

```bash
npx shadcn@2.8.0 add "https://moleculeui.design/r/swipe-row"
```

<Step>Import component</Step>

```tsx
import { SwipeRow } from "@/components/ui/swipe-row"

export default function App() {
  return (
    <div>
      <SwipeRow />
    </div>
  )
}
```

</Steps>
